Build a set of classes / functions which can price a basic consumer loan given the following inputs:

Input Arguments

  • Principal
  • Terms in Months
  • APR - Annual Percentage Rate
  • Lending Date
  • Repayment Day of Month

Output Result

  • Annuity Amount
  • Future Value

Equation

FV = P * \frac{R(1 + R)^N}{(1 + R)^N - 1}

  • FV - Future Value (Annuity Payment)
  • P - Principal
  • R - is the period interest rate
  • N - is the total number of payments
